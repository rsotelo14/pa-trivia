Documentación del Proyecto de Trivia

Descripción

Este proyecto es un juego de trivia implementado en Python que permite al usuario responder preguntas de opción múltiple.

Instrucciones de Uso

Para ejecutar el programa desde la terminal, use el siguiente comando:

python3 trivia.py

El programa mostrará una pregunta junto con opciones posibles. El usuario debe ingresar 1, 2, o 3 para seleccionar su respuesta.

Ejemplo de Interacción


¿Cuál es la moneda oficial de Japón?
1. "Yen"
2. "Dólar"
3. "Euro"
Ingrese su respuesta: 1

Estructura del Proyecto

  • trivia.py: Archivo principal que contiene la lógica del juego de trivia.
  • trivia_questions.csv: Archivo CSV que contiene las preguntas y respuestas del juego.
  • README.txt: Archivo de documentación que describe el funcionamiento del proyecto.

Documentación de Funciones

  • check_file_exists: Decorador que verifica si el archivo especificado existe antes de ejecutar la función decorada. Si el archivo no se encuentra, se muestra un mensaje de error.
  • load_data(file_path): Carga los datos desde un archivo CSV. Abre el archivo trivia_questions.csv, lee las preguntas, las mezcla aleatoriamente y devuelve una lista de preguntas.
  • process_questions(q_and_as, amount=5): Procesa las preguntas y devuelve un generador que presenta cada pregunta, sus opciones, la respuesta correcta y el índice de la respuesta correcta.
  • show_question(q_and_a): Muestra una pregunta y sus opciones en la consola, permitiendo al usuario ver las posibles respuestas.
